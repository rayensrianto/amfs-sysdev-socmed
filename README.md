## Run Locally

Clone the project

```bash
  git clone https://github.com/rayensrianto/amfs-sysdev-socmed.git
```

Go to the project directory

```bash
  cd amfs-sysdev-socmed
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
